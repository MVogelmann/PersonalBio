Computing Biography

Within the past 5 semesters, I learned several programming languages and the use of specific computer software.

  • C/C++: very good knowledge. In Germany about 80 percent of the graduated Electronic Engineers will be employed in an industrial environment which requires very good knowledge of C/C++. Especially the field of Embedded Systems growed very important during the last decades. Therefore, my German home university taught me during the first three semesters a lot about C and also C++. Due to my internships and my non-term jobs in the automotive branch I also got to know the practical aspect.
  • Assembling: good knowledge. As a basic programming language, the assembling language is still today very important. Especially when the code size is strongly limited or an existing high-language code has to be optimized.
  • Matlab
  • VHDL
  • LabVIEW

Even if my major includes a very good education in Embedded Systems languages, we learn neither HTML, CSS nor JAVA at all.
